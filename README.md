# wavevis.aolabs.io

Vite + React + TypeScript tool for programming a 2D grid of double-layer Sarrus cells and inspecting the generated 3D mechanism.

## Commands

- `npm run dev`
- `npm run build`
- `npm run preview`
- `npm run deploy`

The public build uses relative asset paths so the same `dist/` output works at `https://wavevis.aolabs.io/` and the temporary `https://aolabs.io/wavevis/` fallback.
